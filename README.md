# Список задач (ToDo) на React

**Stack:**
- ReactJS (useState, useReducer, useEffect)
- React Router
- Axios
- classnames
- FontAwesome
- SweetAlert2
- JSON-server

### Installation

Requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ npm i
$ npm run json-server
$ npm run start
```
